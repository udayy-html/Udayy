Website sederhana ini dibuat dengan HTML, CSS, dan JavaScript

Belajar HTML

HTML adalah bahasa dasar untuk membuat struktur website. Dengan HTML, kita dapat membuat teks, gambar, tabel, dan banyak elemen lainnya.

Belajar CSS

CSS digunakan untuk mempercantik tampilan website, mengatur warna, font, jarak, dan layout halaman.

Belajar JavaScript

JavaScript menambahkan interaktivitas pada website, seperti membuat tombol bisa diklik, animasi, dan banyak lagi.
